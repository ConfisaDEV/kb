Para cadastrar a carteira, acesse o módulo 'Cadastro de Contas' através do caminho '<Cadastro\Operações Financeiras\Cadastro de Contas>', ou digite a tecla de atalho 'CTRL + P' e pesquise pelo termo 'Conta' ou 'Contas'. Se já possui a conta bancária cadastrada, selecione-a clicando no botão 'Pesquisar'. Caso ainda não possua a conta bancária cadastrada, clique aqui.

Após cadastrar a carteira, faça uma remessa de cobrança contendo entre 2 a 3 contas, geradas como teste. Para saber como gerar o arquivo de remessa, consulte este artigo.


Com o arquivo gerado, deve-se enviá-lo ao banco para validação. Se a validação retornar sem erros, indique na carteira o campo 'Homologado' como 'Sim'. Após a homologação, pode-se gerar os boletos para os clientes.
