Neste artigo veremos como preencher os dados no cadastro de CFOP para emissão da nota fiscal.


Esses dados são: Situação de ICMS, Situação de IPI, Enquadramento de  IPI, Situação de PIS e Situação de COFINS.


O código do 'Enquadramento de  IPI' deve ser verificado junto ao seu Contador conforme a operação da sua empresa, se não estiver compatível o código do 'Enquadramento de  IPI' com a operação pode ocorrer Rejeições ao validar a NF-e no Sefaz.

O CFOP é um código fiscal que identificará a operação de circulação da mercadoria. Esses dados devem ser fornecidos pelo escritório contábil, e toda a informação passada pelo escritório é de responsabilidade do usuário preencher no sistema.

Para acessar o cadastro de CFOP basta seguir o caminho <Cadastros\Operações Fiscais\CFOPs>.

Figura 1: Acessando o cadastro de CFOP.


Todos os CFOPs já estão previamente cadastrados e com algumas informações já preenchidas, tais como: Local do estado, Tipo CFOP, Tipo destinatário, Origem do produto. Basta apenas clicar no campo 'Pesquisa' e digitar pelo código do CFOP que deseja. Após localizar, preencha os campos de tributação como na simulação abaixo:  

Figura 2: Configuração de CFOP.


No campo 'Gera Financeiro', marcar 'Sim' para que ao fazer a nota fiscal seja possível criar as contas a receber da nota, após ter sido transformada de uma fatura ou feita manualmente .

Se for um CFOP de compra, deve também ser marcado o campo 'Gera Financeiro' como 'Sim', para que o sistema crie as movimentações financeiras daquela compra a ser paga, tanto vinda de um recebimento de compras quanto feita manualmente. 

Para que uma nota possa calcular ICMS para empresas simples, ela deve usar uma situação tributária que determine esse cálculo. Como exemplo cita-se o código 102 - Tributado Sem Permissão de Crédito, que deve ser configurado no campo 'Situação ICMS'. 

Caso seja configurado um código de isenção, como 400 por exemplo, o sistema não fará esse cálculo.

Essa regra vale também para o IPI, PIS e COFINS, que possuem seus próprios códigos de tributação e isenção, como 050 para cálculo de IPI e 053 para isenção. Os códigos a serem utilizados devem ser obtidos com seu contador.

Qualquer código utilizado sem a devida consulta à contabilidade pode gerar problemas na arrecadação de impostos e outros problemas com a Receita Federal.


Resumo

Acesso ao Cadastro de CFOP

O cadastro de CFOP pode ser acessado pelo menu "Cadastros", em "Operações Fiscais" e, em seguida, "Cadastro de  CFOP".

Alternativamente, é possível clicar em "Pesquisar Módulo" e digitar "CFOP" para localizar rapidamente o cadastro.

O sistema já vem com todas as CFOP existentes cadastradas por padrão, permitindo a busca pelo código da CFOP.


Funcionalidades da Tela de Cadastro

Na tela do cadastro, é possível pesquisar por nome ou código da CFOP.

Um botão em formato de “olho” permite visualizar mais opções, como campos ocultos que não são frequentemente  utilizados.

É viável configurar detalhes como se a CFOP gera financeiro (sim ou não), que indica se uma nota deve ou não gerar contas a receber ou a pagar; Destino (dentro ou fora do estado), e se é uma entrada ou saída.


Configurações Específicas

Também é possível especificar o tipo de pessoa (física, jurídica ou ambas), para o qual a nota será emitida.

Há um campo para indicar a origem da CFOP e espaço para observações livres sobre seu uso.

Opções adicionais incluem validação de situação tributária e configuração para diferentes finalidades como venda, compra ou devolução.


Personalização por Filial

Configurações específicas podem ser feitas por filial; por exemplo, na CFOP 5101.

Se houver mais de uma empresa, é possível definir situações tributárias específicas para cada uma delas.

Devemos informar as situações tributárias de ICMS, IPI, PIS e Cofins. Essas informações devem ser usadas para emissão da nota fiscal.


Funções Adicionais

Após realizar alterações no cadastro, basta clicar em salvar; também existe a opção de excluir uma CFOP
