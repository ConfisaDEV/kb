Neste artigo veremos como realizar o cadastro de um cliente.


Acesse o módulo de cadastro de clientes através do menu <Cadastros\Empresas e Pessoas\Clientes> ou digite as teclas de atalho 'CTRL + F' e busque pelo termo de pesquisa 'clientes'  ou  'cadastro'. Para inserir um novo cliente, clique no botão 'Adicionar' e digite seus dados de cadastro, como documento, endereço, telefone etc. 

Figura 1: Cadastro de cliente - Aba dados cadastrais.


*Importante: Não utilize caracteres especiais nos campos do endereço (tais como: asteriscos(*), traços(-), cerquilhas(#), etc.), pois podem gerar problemas na validação de notas fiscais ou arquivos de remessa bancária. 


Também não deve ser indicada Razão Social que supere a quantidade de 60 caracteres.


Para inserir o endereço, basta digitar o CEP que o sistema preencherá as informações automaticamente, sendo necessário indicar apenas o número do endereço. Observe nas imagens abaixo:

Figura 2: Inserindo endereço pelo CEP.


Caso a cidade não possua CEP por Rua, indique a rua manualmente:

Figura 3: Inserindo endereço pelo CEP quando não usa CEP por rua.


Observação: Caso apareça uma mensagem indicando que o CEP não foi encontrado, verifique com o cliente se o número realmente está certo, pois o sistema utiliza o IBGE como fonte de validação para o CEP.


Na aba 'Dados comerciais' podemos indicar um representante e uma transportadora padrão para o cliente, dessa forma serão automaticamente vinculados nos pedidos. Nessa aba também Podemos indicar a qual 'Grupo de pessoas' esse cliente pertence.

Figura 4: Cadastro de clientes - Aba dados Fiscais.


A aba 'Observação' nos permite indicar uma observação daquele cliente. Na parte inferior da tela podemos indicar uma mensagem para ser exibida quando fizermos uma fatura para esse cliente.

Figura 5: Cadastro de clientes - Aba observação.


A próxima aba refere-se aos 'Dados Financeiros' do cliente, como por exemplo uma tabela de preço, condição de pagamento específica, realizar consultas de crédito, etc.  bem como podemos bloquear e desbloquear o cadastro desse cliente. Para saber mais sobre o bloqueio de clientes, clique aqui.

Figura 6: Cadastro de clientes - Aba dados financeiros.


Na aba 'Dados fiscais' podemos indicar informações relacionadas à contabilidade, como por exemplo um e-mail padrão para envio de XML e danfes, situações tributárias do ICMS daquele cliente, etc.

Temos ainda a aba 'Complementos', onde temos a possibilidade de informar motivos de pendências financeiras que podem ser motivo para bloqueio de vendas para ele.

Resumo

Acesso ao Módulo de Cadastro de Clientes

Para iniciar o cadastro de um cliente, devemos acessar o módulo de Cadastro de clientes, que pode ser feito pelo menu "Cadastros" e depois "Empresas e Pessoas". Podemos acessar também através da opção “Pesquisar módulos” e digitando "Cliente".

Ao clicar em "Adicionar", é possível cadastrar um novo cliente informando a razão social ou nome do cliente.

Informações Básicas do Cliente

Durante o cadastro, é necessário escolher entre pessoa física, jurídica ou estrangeira. Para pessoas físicas, o CPF deve ser inserido e validado.

Podemos incluir os clientes em grupos, para isso basta clicar na aba “Dados comerciais” e informar os grupos desejados. 
Além do CPF, informações como RG e data de nascimento podem ser adicionadas. Para empresas, há uma opção para
consultar dados automaticamente via Sefaz.

Obs: Porém é importante salientar que para consultar dados na Sefaz, é preciso utilizar o certificado digital da empresa para buscar as informações.

Endereço e Contatos

O sistema permite adicionar endereços automaticamente ao digitar partes do endereço ou o CEP. É possível incluir múltiplos endereços para cobrança ou entrega.

Também é viável adicionar diversos tipos de telefone e até integrar com WhatsApp Web para facilitar a comunicação.

Dados Financeiros e Limites

Na aba financeira, é possível definir limites de crédito para o cliente. Ao informarmos um valor como limite, o sistema verificará no pedido de venda se esse valor foi ultrapassado, exibindo uma mensagem com essa informação.

É possível configurar uma condição de pagamento como padrão, dessa forma ao iniciar uma venda o sistema trará essa condição de pagamento automaticamente.

Consulta de Crédito

O sistema permite registrar ocorrências financeiras manualmente, como “Nome sujo na praça”. Podemos ainda integrar o sistema com serviços como Serasa ou Boa Vista para consultas automáticas.

É possível marcar pendências diretamente no sistema após consulta externa.

Complementos e Observações Finais

Informações adicionais podem incluir porcentagens de desconto por antecipação no pagamento.

No cadastro fiscal, detalhes sobre isenções tributárias específicas, como Suframa, podem ser configuradas para emissão de nota.

Cadastro de Clientes e Funcionalidades do Sistema

Importância do Cadastro de Clientes

Para emitir uma venda, é preciso primeiramente cadastrar o cliente, dessa forma o sistema lançará as contas a receber para o devido cadastro.

O cadastro de clientes também permite que a nota fiscal seja emitida para o cliente com os dados corretos, mantendo um cadastro unificado do mesmo.

O sistema permite a exclusão de clientes, caso não tenham sido utilizados ainda.

Pode-se também fazer a impressão de um relatório de clientes, o qual contém dados diversos para conferência e correção.

É possível adicionar e atualizar dados do cliente, incluindo CNPJ para pessoas jurídicas, caso haja alterações.

Funcionalidades Adicionais

Podemos informar um representante padrão, além de uma transportadora.

É possível também informar uma tabela de preço padrão para esse cliente, além de condições de pagamentos não permitidas, bloqueando que sejam utilizadas com ele.
