Acesso ao Cadastro de Alíquotas do Sat

Para cadastrarmos alíquotas do Sat, devemos acessar o módulo “Cadastro de alíquotas do Sat” clicando no menu "Cadastros", seguido por "Operações fiscais" e, finalmente, "Alíquotas". Alternativamente, é possível pesquisar diretamente pelo módulo digitando "Sat" na barra de pesquisa. Ao encontrar o módulo, deve-se dar dois cliques para abrir.

 

Alíquotas Padrão e Cadastro de Novas Alíquotas

O sistema já vem pré-cadastrado com três alíquotas: (FF) - substituição tributária, ( I) - Isento e (NN) Não-tributado. Para cadastrar uma nova alíquota, utiliza-se a setinha para baixo e insere-se um código (exemplo: 18), seguido da descrição que referencia essa alíquota (exemplo: ICMS 18%).

 

Detalhes sobre as Alíquotas

No campo destinado à alíquota, deve-se inserir a porcentagem correspondente (por exemplo: 18% para ICMS). Também é possível adicionar uma segunda alíquota como ISS com 8%. Para cadastrar alíquotas não-tributadas, pode-se usar um código qualquer, como 03, com a porcentagem zero e selecionar a opção "Não tem alíquota".
