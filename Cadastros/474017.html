Neste artigo veremos como cadastrar um Subgrupo de produtos.


O Subgrupo de produtos serve para categorizar os produtos permitindo obter relatórios utilizando filtros por classificação de produtos. ( ex. relatório de vendas por Grupo e Subgrupo de produtos). 


Para acessar o módulo de cadastro de subgrupos, acesse o menu <Cadastros\Itens/Produtos\Especificações dos Produtos\Subgrupos> ou digite as teclas de atalho 'CTRL + F' e procure pelo termo de busca 'subgrupo':

Figura 1: Cadastro de Subgrupo.


Para incluir um novo Subgrupo, clique no botão 'Incluir' e indique uma descrição. É possível definir uma porcentagem Máx. de desconto ou acréscimo para aquele subgrupo nos campos '% Max Desconto' e '% Max Acréscimo'. Para habilitá-los, devemos clicar no botão que exibe campos ocultos, como indicado abaixo:
