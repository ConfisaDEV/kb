Neste artigo veremos como cadastrar um novo produto no Confisa.


Na menu lateral esquerdo, acesse o módulo através do caminho <Cadastros\Itens / Produtos\Produtos>, como demonstra a figura a seguir:

Figura 1: Acessando o módulo CAD0030 - Cadastro de Produtos.


Ao selecionar a opção 'Produtos', uma nova janela será exibida na tela. Nessa janela, serão listados os produtos cadastrados atualmente. Para cadastrar um novo produto, selecione o botão 'Adicionar'. 


Aba 'Dados básicos'

Nesta aba ficam as informações principais do item. É obrigatório informar os campos 'Descrição' e 'Unidade'. Se o seu produto usa Cor e/ou Tamanho, também será obrigatório informá-los no ato do cadastro. O mesmo se aplica para produtos que possuem código manual, sendo necessário digitar o código do item:

Figura 2: Cadastrando um novo produto.


Também podem ser informados os campos 'NCM', 'Preço de Venda' e 'Preço de Custo' e os códigos de barras.


É possível utilizar a opção 'Copiar e Colar' para adicionar um novo produto. Se o seu produto utiliza Cor e/ou Tamanho, as cores e a grade serão adicionadas à esse novo item. Marque como Inativo as cores e/ou tamanhos que não serão utilizados nesse produto:
 

Figura 3: Cadastrando um produto através do botão Copiar e Colar.


Resumo

Acesso ao Cadastro de Produtos que imprimem etiquetas para gôndolas
Para acessar o menu de cadastro de itens de produtos complementares, especificamente para impressão de etiquetas. O caminho no sistema é: "Cadastros\Itens de produto complementares\Etiquetas impressões". É possível buscar diretamente pelo produto digitando seu nome na pesquisa, facilitando a localização.


Seleção dos Produtos para Etiquetas
O cadastro é utilizado para selecionar apenas os produtos que necessitam da troca de etiqueta nas gôndolas, caso haja alteração de preço, evitando a impressão desnecessária. Ao selecionar os produtos desejados, o sistema traz automaticamente os códigos de barras desses itens.


Alterações e Reimpressão das Etiquetas
Quando um produto tem seu preço alterado ou é comprado novamente para repor estoque, essa mudança faz com que o sistema mostre o produto para imprimir etiqueta de gôndola novamente, dessa forma é possível ter controle para substituir ela na prateleira.
