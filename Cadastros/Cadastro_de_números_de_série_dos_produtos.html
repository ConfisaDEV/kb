Acesso ao Cadastro de Números de Série

Para acessar o cadastro de números de série através do menu "Cadastros" e "Itens/Produtos" e “Especificações de Produtos”. É possível pesquisar diretamente pelo número de série, que leva ao cadastro específico dos produtos que utilizam essa funcionalidade.

O número de série de produtos é muito utilizado para rastreio de produtos em notas fiscais, sendo importante cadastrar e gerenciar os mesmos de maneira correta.

Figura 1: Acessando Cadastro de Números de Série.


Geração e Disponibilização dos Números de Série
Para gerar uma sequência de números de série para um produto, primeiro é preciso marcar no cadastro do mesmo a opção “Gera número de série”, que fica na aba “Complementares”, subaba “Diversos”.  

Selecionando o produto desejado e no botão “Visualizar séries”, é possível gerar múltiplos números de série (Ex: 100 números), definindo um intervalo desejado e utilizando a opção “Ações” e "Gerar números de séries".

Após gerar os números, é necessário marcar a sequência de números gerados como disponíveis para uso em vendas. Para isso, use o botão “Ações” e depois “Disponibilizar números de série”.

Figura 2: Gerar números de série.

Informações Adicionais e Transferência
O sistema permite inserir um número do Inmetro correspondente a cada número de série, essencial para rastreamento, no campo. Podemos clicar no botão “Ações” e escolher a opção “Informar o número de do Inmetro no número de série posicionado”. 
É possível ainda transferir uma sequência de números de série digitando a mesma no módulo e escolhendo a opção “Transferir números de série para outro item” no botão “Ações”. O sistema irá abrir uma janela para seleção do produto para onde a sequência será enviada.
