Neste artigo veremos quais são as unidades de medida permitidas para a emissão de nota fiscal.


Acesse o módulo 'Cadastro de Unidades' em <Cadastros\Itens/Produtos\Especificações de Produtos\Unidades>.

Figura 1: Acessando o módulo de unidades.


Todos as unidades permitidas para a emissão da nota fiscal já vêm pré-cadastradas, e são identificadas pela aba 'Permitida em NF-e', ou seja, não é possível utilizar outras unidades além das marcadas nesse campo para a emissão de nota fiscal. porém é possível criar unidades novas apenas para controle interno.

Figura 2: Cadastro de Unidades.


Resumo

Acesso ao Módulo de Cadastro de Unidades
O cadastro de unidades pode ser acessado pelo menu "Cadastros\Itens e Produtos\Especificações do Produto". Alternativamente, é possível usar o atalho “Ctrl + F” e digitar "Unidade". O sistema já vem com uma grande quantidade de unidades cadastradas, sendo desnecessário cadastrar novas a menos que se trate de uma unidade diferenciada.

Adicionando Novas Unidades
Para adicionar uma nova unidade, clique no botão "Adicionar" onde será possível inserir um código e descrição. É possível visualizar campos adicionais que estão ocultos, clicando no botão “Ocultar” que tem um ícone de um olho. Uma das opções é marcar se a unidade é do tipo externa, o que limita seu uso apenas para recebimento de produtos.

Configurações da Unidade
Se a opção para estoque fracionado for marcada, isso significa que a unidade não poderá ter movimentação em números fracionários (Ex: não é permitido vender 1,5 conjuntos).
