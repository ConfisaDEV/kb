Acesso ao Módulo de Cadastro de Ocorrências para OP

Para acessarmos o módulo de cadastro de ocorrências para OP, Podemos clicar no menu "Produção" sob a opção "Complementares", seguido por "Ocorrências para OP". Alternativamente, é possível clicar em "pesquisar módulos" e digitar "ocorrência" para localizar rapidamente o módulo desejado. Após encontrar, deve-se dar dois cliques para abrir o módulo.

Figura 1: Acessando o cadastro de ocorrências para OP.


Ocorrências Pré-Cadastradas
O sistema já apresenta algumas ocorrências pré-cadastradas, incluindo: Início do expediente, Fim do expediente, Início da produção e Fim da operação. Essas ocorrências são fundamentais para registrar eventos importantes durante a produção.

Tipos de Ocorrências
No campo tipo, é possível especificar se a ocorrência é de início ou parada da produção. Também se podemos definir se a ocorrência é geral (afetando toda a produção) ou específica (relacionada a um processo particular).
