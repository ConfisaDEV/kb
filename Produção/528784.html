Neste artigo veremos como o sistema Confisa ERP permite atualizar os preços de venda e de custos automaticamente, caso ativemos uma ficha.


No Confisa ERP podemos atualizar os preços de venda e de custos de produtos várias maneiras, indo desde finalizar uma ordem de produção até entrada de notas fiscais para matérias-primas. 


Para produtos produzidos, podemos também atualizar os preços ativando uma ficha técnica. Isso permite que os preços no cadastro dos produtos estejam sempre atualizados conforme a ficha ativa no momento.

Quando é clicado para ativar a ficha técnica aberta, o sistema pergunta se desejamos atualizar os preços de venda e de custos; Se clicarmos em 'Sim', os preços do cadastro serão ficarão iguais aos da ficha. Se 'Não', serão mantidos como estão.

Figura 1: Atualizar preços de venda e de custos ativando a ficha técnica.


Caso seja feito o preço de custo diferente por cor, devemos habilitar as configurações 'Diferenciar preços por cor na tabela de preços'  para que o sistema grave no histórico de custos os valores para cada cor separadamente.

Figura 2: Alterando configuração.


Ao clicar para gravar o preço de custo do produto, o sistema gravará no cadastro do mesmo o valor para cada cor contida na ficha técnica.
