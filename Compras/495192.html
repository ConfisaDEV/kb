Neste artigo veremos como fazer um recebimento parcial de compras usando o módulo de conferência de Recebimento de Compras.


Primeiro, é necessário habilitar a configuração que gera um outro pedido para itens restantes do recebimento. Acesse o menu de filiais pelo caminho <Configurações\Cadastro de Filiais/Empresas>. Depois, procure pela aba 'Configurações':

Figura 1: Acessando as configurações da filial.


Procure pela configuração 'gerar pedido de compra restante', clicando na coluna 'Descrição' e pressionando a tecla 'F3'. No campo 'Valor', deixe como 'Sim':


Dessa forma, quando efetuarmos a conferência parcial, poderemos efetuar o recebimento parcial do pedido:

Figura 2: Efetuando a conferência parcial do pedido.


Ao executar o recebimento, podemos escolher a opção 'Recebimento parcial'. O sistema exibirá uma mensagem para gerar outro pedido para os itens restantes:

Figura 3: Efetuando recebimento parcial.

Resumo

Acessando o módulo de Conferência de Produtos para recebimento de compra
Para acessar a funcionalidade de conferência de produtos para recebimentos, devemos clicar no menu "Compras" e a seguir no módulo citado.
É possível também buscar pelo módulo, pressionando as teclas “Ctrl + F” e digitando "conferência" na barra de busca, onde aparecerá o módulo para seleção.

Seleção e Busca do Pedido
Ao acessar a tela de conferência, são exibidos todos os pedidos em aberto que ainda não foram conferidos. Podemos digitar também o código do pedido desejado.

Métodos de Conferência
Existem duas maneiras de realizar a conferência: manualmente, inserindo as quantidades desejadas no campo “Quantidades a receber”, ou utilizando um leitor de código de barras. O uso do leitor reduz significativamente erros como recebimento incorreto de itens (cores ou tamanhos errados).

Processo com Leitor de Código de Barras
Para usar o leitor, deve-se passar o código nas peças individualmente; O sistema registra automaticamente o item conferido. À medida que os itens são conferidos, o sistema atualiza as quantidades automaticamente.

Finalização da Conferência
Após conferir os itens, é possível clicar em "Conferir Parcial" se apenas parte dos itens do pedido de compra está sendo recebida. Se todo o pedido de compra tiver chegado à empresa, após conferir todas as quantidades basta clicar em “Conferir Total”.
Se ocorrer algum erro durante a conferência, é possível cancelar a mesma e reiniciar o processo.

Impressão e Importância da Conferência
Uma vez finalizada a conferência corretamente, pode-se salvar as alterações e gerar um relatório impresso com os itens conferidos.
