Neste artigo veremos como utilizar o módulo 'COM0005 - Recebimento de Compra' quando se usa lotes no sistema.


O recebimento de compras usando lotes é semelhante ao procedimento feito quando não se usa lotes, a diferença está na parte para indicar o lote dos itens. Para mais informações sobre o recebimento sem lotes, clique aqui. 


*Importante: Verifique se o estoque que será utilizado no recebimento gera lote. Para isso, acesse o menu <Estoques\EST0001 - Cadastro de Estoques> e procure pelo estoque. Confira na imagem abaixo:

Figura 1: Conferindo se o estoque selecionado no recebimento gera lote.


No módulo 'COM0005 - Recebimento de Compra', após inserir os itens da compra e suas quantidades, pode-se clicar duas vezes no campo 'Tem Lote' para escolher os lotes de cada produto. Caso clique-se em 'Salvar' sem escolher os lotes primeiro, o sistema trará essa a janela de seleção automaticamente.

Figura 2: Acessando o menu de lotes dos itens no recebimento.


Observe na figura acima que um novo lote já vem pré-adicionado (veja os campos 'Dt. Fabricação', 'Dt. Vencimento' e 'Qtde Considerar' preenchidos), bastando preencher o campo 'Descrição do Lote' para cadastrar um novo lote para o item. 

Caso os lotes selecionados/cadastrados estejam incorretos, pode-se editar as informações clicando em duas vezes no campo 'Tem Lote' para retornar ao menu de 'Cadastro de Lotes'. Observe que a caixa de seleção no campo 'Escolhido previamente' estará selecionada.
