Veja neste artigo como configurar a integração entre o Confisa ERP e um site de e-commerce.


Abra o módulo pelo menu <Faturamentos\Vendas Online/Externas\Configurar Integração de Vendas Online>.

Clique no botão 'Adicionar', e preencha o campo 'Nome da integração' conforme o site que você contratou. Marque o campo 'Ativo' como 'Sim'.

Clique na lupa ao lado do campo 'Cód. Módulo' para selecionar a integração contratada (Loja Integrada, Moovin, Magento, etc.).

Figura 1: Configurações.


Preencha todos os campos com os dados informados pelo site, conforme exemplo da 'Figura 1' acima. Para testar a conexão, clique no botão 'Desconectado', dessa forma o sistema tentará fazer a comunicação com o site, alterando o botão para 'Conectado', indicando que o procedimento foi bem sucedido.


Aba 'Configurações':

  • 'Estoque': Determina de qual estoque o sistema irá baixar os produtos quando fizer um faturamento proveniente de venda online.
  • 'Representante': Insira neste campo o representante que será responsável pelas vendas do site, gerando suas comissões conforme configurado no cadastro de representantes.
  • 'Tab. Preço': Caso este campo seja preenchido, o sistema irá enviar ao site os preços de venda conforme a tabela de preço indicada. Se não for informada nenhuma tabela de preço, o sistema enviará os preços conforme o cadastro de cada produto.
  • 'Promoção': Este campo permite enviar ao site uma lista de preços específica para promoções, diferente da tabela de preços padrão.


Aba 'Status dos pedidos no site':

 

Esta aba informa quais status de pedido existem no site. Nessa tela você pode marcar qual status permite que o site envie pedidos ao sistema.

Exemplo: Se você marcar o status 'Pedido Pago', você só receberá pedidos no sistema quando ele já estiver pago no site.
 

  • Aba 'Cores e tamanhos (Grade)'

Esta aba serve para informar se os produtos têm cor e para informar a grade de tamanhos.
