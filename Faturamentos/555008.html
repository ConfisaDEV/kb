Neste artigo veremos como utilizar o módulo de ponto de venda (PDV) do sistema Confisa ERP.


Para acessar o módulo, clique em <Faturamentos\Confisa PDV>.


Para iniciar uma venda, pressione a tecla 'Enter' no seu teclado.


Será exibida a tela abaixo, onde você irá inserir os produtos para venda:

Figura 1: Inserir produtos.


Para inserir o produto, digite o código do produto e pressione a tecla 'Enter'. Digite a quantidade e pressione 'Enter'.


Caso você queira pesquisar o produto, pressione 'Enter', clique na palavra 'Descrição' e digite o nome do produto, conforme exemplo abaixo:

Figura 2: Pesquisa por nome.


O sistema Confisa ERP é compatível com leitores de código de barras, basta passar a leitora no código de barras do produto e o sistema adicionará ele automaticamente no ponto de venda.

Se precisar inserir o mesmo produto várias vezes na venda, basta digitar 'X' mais a quantidade de produtos no campo 'produtos' para ativar o multiplicador. Passe a leitora no código de barras do produto, que o sistema irá adicionar essa quantidade automaticamente, não necessitando assim colocar o mesmo produto várias vezes no PDV.

Para finalizar a venda, pressione a tecla 'F8'.

Nessa tela é possível acrescentar um desconto por valor ou porcentagem.

Figura 3: Desconto.


Pressione 'Enter' duas vezes para prosseguir.

Na tela que aparece, informe o valor que foi pago e aperte a tecla 'F8' para confirmar.

Figura 4: Recebimento.


É possível utilizar vários meios de pagamento, até mesmo mais de um de uma só vez!

Obs: O sistema disponibiliza algumas teclas de atalho que podem ser utilizadas para deixar seu processo ainda mais rápido.
 

Figura 5: Teclas de atalho.


Pronto, sua venda está concluída!

Resumo

Acessando o módulo Confisa PDV

Para acessar o Confisa PDV, devemos clicar no menu “Faturamentos” e selecionando a opção "Confisa PDV".

Para iniciar uma venda, devemos pressionar a tecla “Enter” acessar a primeira tela da venda no sistema. É possível configurar o sistema para trazer o cliente “Consumir” como padrão, além da forma de pagamento à vista, pulando a necessidade da escolha de cliente no início.

Realizando Vendas
Para adicionar produtos, podemos utilizar o leitor de código de barras, que automaticamente lança o item na venda mostrando o mesmo. Pressionando a tecla “Tab” ajuda na navegação entre os campos sem o uso do mouse.
Caso seja necessário, é possível digitar o nome ou código do produto. O sistema trará as opções cadastradas e permite selecionar o item desejado manualmente.
Após adicionar um item, caso ele tenha sido lançado erroneamente, podemos retirar o produto da venda pressionando “F7” ou clicando em "Cancelar itens". Após escolher o produto a ser cancelado, basta pressionar “F8” para confirmar a retirada.

Consultas e Funcionalidades Adicionais
Durante a venda, se um cliente perguntar sobre outro produto, podemos pressionar a tecla “F4” para consultar preços de produtos usando o leitor de código de barras ou selecionando o item manualmente. Também é possível utilizar uma balança para produtos vendidos por peso; Basta pressionar “F2” enquanto o produto estiver na balança para fazer a leitura do peso e lançar no sistema.

Finalizando Vendas
Para finalizar a venda, pressione “F8”. É possível aplicar descontos antes da confirmação da finalização da venda. O sistema permite lançar a quitação da venda em diferentes formas (Dinheiro, pix ou crédito) simultaneamente. Após finalizar, uma impressão da venda é impresso automaticamente.

Iniciando Novas Transações
Após concluir uma venda, pressione “Enter” para iniciar uma nova transação, agilizando a venda no sistema sem o uso de mouse.
