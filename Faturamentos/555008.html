Neste artigo veremos como utilizar o módulo de ponto de venda (PDV) do sistema Confisa ERP.


Para acessar o módulo, clique em <Faturamento\Confisa PDV>.


Para iniciar uma venda, pressione a tecla 'Enter' no seu teclado.


Será exibida a tela abaixo, onde você irá inserir os produtos para venda:

Figura 1: Inserir produtos.


Para inserir o produto, digite o código do produto e pressione a tecla 'Enter'. Digite a quantidade e pressione 'Enter'.


Caso você queira pesquisar o produto, pressione 'Enter', clique na palavra 'Descrição' e digite o nome do produto, conforme exemplo abaixo:

Figura 2: Pesquisa por nome.


O sistema Confisa ERP é compatível com leitores de código de barras, basta passar a leitora no código de barras do produto e o sistema adicionará ele automaticamente no ponto de venda.

Se precisar inserir o mesmo produto várias vezes na venda, basta digitar 'X' mais a quantidade de produtos no campo 'produtos' para ativar o multiplicador. Passe a leitora no código de barras do produto, que o sistema irá adicionar essa quantidade automaticamente, não necessitando assim colocar o mesmo produto várias vezes no PDV.

Para finalizar a venda, pressione a tecla 'F8'.

Nessa tela é possível acrescentar um desconto por valor ou porcentagem.

Figura 3: Desconto.


Pressione 'Enter' duas vezes para prosseguir.

Na tela que aparece, informe o valor que foi pago e aperte a tecla 'F8' para confirmar.

Figura 4: Recebimento.


É possível utilizar vários meios de pagamento, até mesmo mais de um de uma só vez!

Obs: O sistema disponibiliza algumas teclas de atalho que podem ser utilizadas para deixar seu processo ainda mais rápido.

Figura 5: Teclas de atalho.


Pronto, sua venda está concluída!
