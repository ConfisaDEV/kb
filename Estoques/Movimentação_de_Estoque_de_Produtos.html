Acesso ao módulo de Movimentação de Estoque de Produtos

Para iniciar o acesso ao módulo, clique no menu "Estoques", depois "Movimentações" e selecione "Movimentação de produto". Alternativamente é possível pressionar as teclas “Ctrl + F” e digitar por "movimentação". Na tela inicial, configure os filtros para selecionar o estoque desejado e o produto do qual deseja-se ver as movimentações. É possível ainda ver o período específico, visualizando os movimentos dentro de uma data determinada.

Figura 1: Acessando a movimentação de estoques de produtos.

Filtragem e Visualização dos Movimentos
Após definir o período e clicar em buscar, o sistema exibirá os movimentos filtrados. Informações detalhadas do produto também estarão disponíveis. O saldo anterior ao período selecionado é mostrado em um campo específico, ajudando à conferência de entradas e saídas caso necessário. Se não houver movimentações antes da data selecionada ou o saldo até o último dia antes dela for zero, o saldo anterior será mostrado como zerado. As operações subsequentes (vendas, produções ou lançamentos manuais) serão lançadas no sistema como crédito e débito.

Figura 2: Pesquisando movimentações.

Ajustes Manuais no Estoque
Caso seja necessário ajustar o estoque manualmente, é possível lançar uma movimentação de modo manual. Para isso, basta clicar em “Lançar movimentação” e ler o código de barras do produto com um leitor de código de barras ou digitar as informações necessárias. É possível lançar créditos ou débitos, alterando a descrição para acerto no estoque refletindo o motivo desse lançamento, e inserir a quantidade desejada.

Exclusão e Relatórios das Movimentações
Se necessário, é possível também excluir uma movimentação desde que o usuário tenha permissão. Ao deletar uma movimentação, o saldo retornará ao valor anterior. Além disso, há opções para imprimir ou visualizar relatórios das movimentações na tela caso deseje um registro físico ou digital.

Registro de Empenho
Caso haja uma ordem de produção empenhando uma quantidade de um material, é possível ver essa informação na aba “Empenho”.
