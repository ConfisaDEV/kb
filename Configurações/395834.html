Neste artigo veremos como realizar o procedimento de envio de produtos para o Marketplace. 


Acesse o menu da integração através do caminho <Sistema\Integrações\Confisa Integração Marketplace> ou digitando as teclas de atalho 'CTRL + P' e pesquisando pelo termo 'marketplace'. No menu, procure pela aba 'Produtos':

Figura 1: Acessando o menu do Marketplace para envio de produtos.


*Importante: O envio dos produtos será efetuado automaticamente a cada 7 minutos quando houver alterações nos campos obrigatórios. Para mais informações sobre os campos obrigatórios, clique aqui. Também deverá ser realizado o envio quando for indicado um novo produto para o canal. Para saber mais sobre produtos de cada canal, clique aqui


Observe o exemplo abaixo, onde será adicionado um novo produto para o canal 'Mercado Livre'. Após indicar o novo produto no canal, ele será listado como um produto pendente para envio. Para listar os produtos pendentes, clique no botão 'Buscar - Listar produtos pendentes de envio':

Figura 2: Listando produtos pendentes de envio.


Observe um outro exemplo onde o preço de um produto já enviado sofre uma alteração em seu preço de venda. Também ficará como pendente para envio:

Figura 3: Produto pendente para envio após sofrer alteração de preço.


Para efetivamente enviar os produtos aos canais, clique sobre o botão 'Enviar Produtos':

Figura 4: Enviando os produtos para os canais.


Após finalizar o envio, o sistema exibirá uma mensagem com o status da operação. Caso exista alguma divergência ou problema no envio, identifique-as pela aba 'Inconsistências':

Figura 5: Inconsistência encontrada: Produto sem a Marca informada no cadastro.


Para checar o envio, acesse o hub do Marketplace a partir do link https://hub.sandbox.plug4market.com.br/. e acesse a listagem de produtos:

Figura 6: Produtos enviados no canal.
