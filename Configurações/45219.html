2) Liberando acesso: 


Alguns computadores Windows exigem que a pasta seja liberada para que o programa consiga sincronizar os dados corretamente. 

Acesse a pasta do sistema em C:\Program Files (x86)\Confisa Sistemas\Confisa Mobile Windows e clique com o botão direito para selecionar a opção 'propriedades', conforme exemplo abaixo:

Figura 2: Permissão na pasta

 

3) Configurando o primeiro acesso ao sistema:

Informe o código do dispositivo (este código você deve solicitar para a empresa da qual é representante).


Informe o código do representante no Sistema Confisa ERP (este código você deve solicitar para a empresa da qual é representante).

Clique em 'Salvar'.
 

Figura 3: Configurações.
