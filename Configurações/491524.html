Veja neste artigo como acessar o portal da Confisa com os dashboards da sua empresa, permitindo a visualização tanto na tela inicial do sistema ou pelo seu celular

Sistema:
- No sistema vá em cadastro de usuários e informe o e-mail que utilizará para acessar.
- Clique em liberar o acesso web para que seja enviada uma requisição para o site.
Pronto! Seu acesso ao portal estará liberado, basta acessar com suas credenciais.

Portal: https://app.confisa.com.br
E-mail: E-mail informado no cadastro de usuário no sistema ERP;
Senha: A mesma senha que você acessa o sistema ERP.

Na tela principal do sistema:
 - Clique no botão Adicionar Dashboard;
 - Irá abrir uma tela para você habilitar os dashboards que o usuário escolhido poderá visualizar.
 - Depois de escolhido os dashboards cliquem em Confirmar.
Importante: Os dashboards habilitados estarão disponíveis para o usuário visualizar em até 15 minutos.

OBS: Caso não tenha permissão de acesso a tela de adição de dashboard, solicite a seu supervisor para executar esse procedimento.
